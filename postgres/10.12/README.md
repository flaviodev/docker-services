# Postgres 10.12

### Running Postgres without volume

````bash

$ ./wihout-volume-run.sh

````

### Copying the Postgres data folder to a volume

````bash

$ ./wihout-volume-run.sh 

$ ./copy-pg-data-to-volume.sh

````

### Running Postgres using a volume

````bash

$ ./run.sh

````

### Execute commands on the container

````bash

$ ./wihout-volume-run.sh

// or

$ ./run.sh


$ ./exec.sh

````
